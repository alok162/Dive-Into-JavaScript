# What are the java script data types

JavaScript supports three Primary, two Composite and two Special data types. Next, we list down the data types in each of the categories.

**Primary Data Types**
  1. String
  2. Number
  3. Boolean

**Composite Data Types**
  1. Object
  2. Array
  
**Special Data Types**
  1. Null
  2. Undefined
